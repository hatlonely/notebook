# mac 分屏工具 rectangle

## 安装

```shell
brew install --cask rectangle
```

## 使用

安装后需要手动运行，运行之后拖动即可分屏

也可以快捷键分屏

- `ctl` + `opt` + `→`：靠右分屏
- `ctl` + `opt` + `←`：靠左分屏
- `ctl` + `opt` + `↑`：靠上分屏
- `ctl` + `opt` + `↓`：靠下分屏
- `ctl` + `opt` + `⏎`：全屏
