# wails 快速入门

## 安装

```shell
go install github.com/wailsapp/wails/v2/cmd/wails@latest
sudo apt install -y libgtk-3-dev
sudo apt install -y libwebkit2gtk-4.0-dev
sudo apt install -y nsis
```

## 创建项目

```shell
wails init -n helloworld -t svelte-ts
```
