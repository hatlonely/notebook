# svelte highlight.js

## 安装

```
npm i -D svelte-highlight
```


## 常见错误

安装 highlight.js 时报错

```
 Cannot read properties of null (reading 'edgesOut')
```

解决方案

```
npm i -D styled-components@5.3.10 @types/styled-components
```
