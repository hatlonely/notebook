## vscode 配置

## 快捷键

windows:

运行命令: `ctrl + shift + p`
快速搜索: `ctrl + p`
打开设置: `ctrl + ，`

## 外观

### 产品主题

1. 打开插件，搜索 MacOS Modern Theme

### 字体切换

1. 安装插件 `Font Switcher`
2. `ctrl + shift + p` 打开命令面板
3. 输入 `switch font` 切换字体

## 终端

### 选中复制

1. 打开设置
2. 搜索 `selection`
3. 勾选 `terminal.integrated.copyOnSelection`

## python

1. 打开插件，安装 Python 插件

> 如果代码中有库缺失，会导致测试插件无法检测到测试用例，手动安装缺失库之后可以解决
